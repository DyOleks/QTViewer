How to build manual

1. Navigate to ShowImages_build folder and run CMake to configure and generate a native build system:
	cd ShowImages_build
	cmake ../ShowImages

2. then call that build system to actually compile/link the project:
	cmake --build .

3. launch the binary
	./ShowImages
